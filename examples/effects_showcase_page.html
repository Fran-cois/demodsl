Simple YAML Config

Describe your demo in YAML. DemoDSL handles the rest — recording, effects, narration, and export.

# One config, infinite demos
metadata:
  title: "My Product Demo"

scenarios:
  - name: "Feature Tour"
    provider: "selenium"
    viewport: { width: 3840, height: 2160 }
    steps:
      - action: "navigate"
        url: "https://myapp.com"
        effects:
          - type: "spotlight"
          - type: "confetti"
